# Zillow-Dashboard-Project

## Overview
Real estate dashboard project using SQL and Power BI. Cleaned, transformed, and visualized home value trends by city and state.

## Workflow
1. Imported CSV into MySQL using the **Import Wizard**, which automatically created the table.
2. Cleaned and transformed the data (unpivoted monthly columns into a long format) using SQL queries.
3. Connected the cleaned table to Power BI Desktop.
4. Built interactive dashboards showing:
   - Average home value trends by city and state
   - Top 5 most expensive cities
   - Month-over-month home value changes
 
 ## Files Included
 - `zillow_sample_100.csv` – Sample dataset
- `analysis_queries.sql` – SQL queries used for cleaning & unpivoting
- `dashboard.pbix` – Power BI dashboard file
- `screenshots/` – Key visuals from the dashboard

## Skills Demonstrated
- SQL: data import, cleaning, unpivoting, and aggregation
- Power BI: interactive dashboards, line charts, bar charts, slicers, and KPIs
- Data analysis: trends over time, top markets, and regional comparisons

  ## How to View
1. Download the `.pbix` file and open it in **Power BI Desktop**
2. Or view the dashboard screenshots in `screenshots/`
